## Auth Reverse Proxy

### Install

```
go install github.com/wiesson/auth-proxy
```

### arguments

Example

```
auth-proxy -host https://api.myawesomecompany.com/v2/ -token my-super-secret-token
```

#### -host*

example: `-host https://api.myawesomecompany.com/v2/`

#### -token*

example: `-token my-super-secret-token`
